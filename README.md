# Kanban Task Board

This application is supported through Vercel and Supabase. It presents a Kanban Board similar to those found on Jira.
Currently supports only anonymous sign in.

# Features
- Adding New Tasks
- Dragging and Dropping Tasks into the different columns (To Do, In Progress, In Review, Done)
- Deleting Tasks
- Sorting Tasks by Priority (Low, Normal, High)
- Sorting Tasks by Search Results
- Summary Bar under the header that displays the total # of tasks, total # of overdue tasks, and total # of completed tasks
